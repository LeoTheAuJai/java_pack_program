# Homework 3 - Simple Database with List

## Purpose
Practice using Java collections (`List`) to manage data. This is a simplified in-memory database implemented with classes.

## Features
- Store and manage data using a `List` structure.
- Basic operations (add, remove, find) are implemented through Java classes.
- No external database; data exists only during program runtime.

## Key Concepts
- Java Collections Framework (`List`, `ArrayList`)
- Class design for data management
- Basic CRUD operations (Create, Read, Update, Delete) in memory

# Homework 3 - 使用 List 的簡易資料庫

## 目的
練習使用 Java 集合框架 (`List`) 來管理資料。這是一個以類別實作的簡易記憶體資料庫。

## 功能
- 使用 `List` 結構儲存與管理資料。
- 透過 Java 類別實作基本操作（新增、刪除、查詢）。
- 無外部資料庫，資料僅在程式執行期間存在。

## 學習重點
- Java 集合框架 (`List`, `ArrayList`)
- 資料管理類別的設計
- 記憶體中的基本 CRUD 操作
