## 🚀 How to Run These Projects

### Using BlueJ (Recommended)
1. Open BlueJ
2. Click **"Open Project"** and navigate to the specific homework/lab folder
3. Find the class with `main()` method (e.g., `MyClock`, `UseBST`, `GUI`)
4. Right-click → **"void main(String[] args)"** → **"Execute"**

### Using Command Line
```bash
# Compile all Java files in a folder
javac *.java

# Run the main class (replace MainClassName with actual class name)
java MainClassName
```

### Using VS Code
1. Install **Extension Pack for Java**
2. Open the project folder
3. Locate the class with `main()` method
4. Click the **"Run"** button above the method

## 🚀 如何執行這些專案

### 使用 BlueJ（推薦）
1. 打開 BlueJ
2. 點擊 **「Open Project」**，進入對應的 homework/lab 資料夾
3. 找到包含 `main()` 方法的類別（例如 `MyClock`、`UseBST`、`GUI`）
4. 右鍵 → **「void main(String[] args)」** → **「Execute」**

### 使用命令列
```bash
# 編譯資料夾中的所有 Java 檔案
javac *.java

# 執行程式（將 MainClassName 替換為實際類別名稱）
java MainClassName
```

### 使用 VS Code
1. 安裝 **Extension Pack for Java**
2. 打開專案資料夾
3. 找到包含 `main()` 方法的類別
4. 點擊方法上方的 **「Run」** 按鈕
